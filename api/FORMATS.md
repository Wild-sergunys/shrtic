# Форматы запросов и ошибок SHRTIC API

## Общий формат запросов

Все запросы с телом принимают JSON. Заголовок Content-Type: application/json обязателен.

## Общий формат успешного ответа

Для операций создания возвращается ID:
{
  "id": 5
}

Для операций обновления/удаления возвращается сообщение:
{
  "message": "Операция выполнена успешно"
}

## Формат ошибки

Все ошибки возвращаются в едином формате:

{
  "error": "validation_error",
  "message": "Поле url обязательно",
  "details": {}
}

### Поля ответа с ошибкой

| Поле | Тип | Описание |
|------|-----|----------|
| error | string | Тип ошибки (код) |
| message | string | Человекочитаемое сообщение |
| details | object | Дополнительная информация (опционально) |

### Типы ошибок

| Код | HTTP | Описание |
|-----|------|----------|
| validation_error | 400 | Ошибка валидации входных данных |
| invalid_credentials | 401 | Неверный логин или пароль |
| unauthorized | 401 | Не авторизован |
| forbidden | 403 | Нет прав доступа |
| not_found | 404 | Ресурс не найден |
| conflict | 409 | Конфликт данных (логин уже занят) |
| rate_limit | 429 | Слишком много попыток входа |
| internal_error | 500 | Внутренняя ошибка сервера |

## HTTP-коды ответов

| Код | Название | Когда используется |
|-----|----------|---------------------|
| 200 | OK | Успешный GET, PUT, DELETE |
| 201 | Created | Успешный POST (ресурс создан) |
| 302 | Found | Перенаправление |
| 400 | Bad Request | Ошибка валидации, конфликт данных |
| 401 | Unauthorized | Не авторизован |
| 403 | Forbidden | Нет прав доступа |
| 404 | Not Found | Ресурс не найден |
| 409 | Conflict | Конфликт данных |
| 429 | Too Many Requests | Превышен лимит попыток входа |
| 500 | Internal Server Error | Ошибка сервера |

## Примеры

### Успешный логин (200)
{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "role": "user"
}

### Ошибка логина (401)
{
  "error": "invalid_credentials",
  "message": "Неверный логин или пароль"
}

### Превышен лимит попыток входа (429)
{
  "error": "rate_limit",
  "message": "Слишком много попыток входа. Попробуйте позже.",
  "details": {
    "blocked_for_seconds": 900
  }
}

### Создание ссылки (201)
{
  "id": 42,
  "short_url": "/r/x7kQ2",
  "long_url": "https://github.com/user/repo",
  "clicks": 0,
  "created_at": "2026-04-26T20:30:00Z"
}

### Ссылка не найдена (404)
{
  "error": "not_found",
  "message": "Ссылка не найдена"
}

### Конфликт при регистрации (409)
{
  "error": "conflict",
  "message": "Пользователь с таким логином уже существует"
}

### Удаление чужой ссылки (403)
{
  "error": "forbidden",
  "message": "Нет прав на удаление этой ссылки"
}
